# Results on SiSEC 2018

Open-Unmix yields state-of-the-art results compared to participants from SiSEC 2018 (opens new window). The performance of UMXHQ and UMX is almost identical since it was evaluated on compressed STEMS. It can be seen that our proposed model reaches state-of-the-art results. There is no statistically significant difference between the best method TAK1 and UMX. Because TAK1 is not released as open-source, this indicates that Open-Unmix is the current state-of-the-art open-source source separation system.

Note that

  1. [STL1, TAK2, TAK3, TAU1, UHL3, UMXHQ] were omitted as were trained on only MUSDB18.
